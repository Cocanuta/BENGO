Bengo 2.0

The backend framework built for the SnatchyBuckles E3 2019 Bingo game. Developed as a companion to the E3 Bingo twitch extension.

Testing Environment

Running on NodeJS first install required packages: npm install.

Then run the development build: npm run dev.

Built With

  • TwitchAPI - The official Twitch API.
  • Express - Used as the web server framework.
  • Socket.io - Used as the websocket framework.
  • SQLite3 - Used for all database storage.
  • JSDoc - Used for generating all documentation.

Versioning

We use SemVer for versioning.
